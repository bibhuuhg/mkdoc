Usage

After installing sparkbatch in your conda environment, there are two steps to use and invoke the packages .

Step 1: invoke using CLI.

$ python -m sparkbatch --help 
$ python -m sparkbatch --version 
$ python -m sparkbatch --input_data_path <input_data_path> --brand <brand_name> --target_dir <output_dir>


where input_data_path : input data location
brand_name : To fetch record for a brand
output_dir : Output location where final result will be stored

example :

python -m sparkbatch --input_data_path "user/bibhu/input/" --brand "clp" --target_dir "/user/bibhu/pyspark_output/"

Step 2: invoke programatically.

from sparkbatch.run import get_data_by_brand
from sparkbatch.spark_utils import write_df_to_target 
df = get_data_by_brand(data_path : str, brand: str)
write_df_to_target(df : DataFrame, target_dir: str )
